# 천재 탐정 K: 방탈출 데모 웹사이트 SPEC 명세서

## 1. 개요 (Overview)
- **주제**: 탐정 K의 비밀 수첩을 테마로 한 웹 방탈출 게임 (행사 데모용)
- **목표**: 2~5분 내외의 플레이 타임, 직관적이고 시각적인 상호작용
- **타겟 디바이스**: 행사장에 비치된 PC (데스크톱/노트북) 환경

## 2. 기술 스택 (Tech Stack)
- **프레임워크**: Next.js (React)
- **스타일링**: Tailwind CSS
- **상태 관리 및 진행 데이터 보존**: React Context / State, 브라우저 LocalStorage 활용 (새로고침 시에도 스테이지 유지)

## 3. 핵심 UI/UX (Core UI/UX)
- **비주얼 노벨 스타일 대사창**: 화면 하단에 고정된 텍스트 박스로 상황 설명 및 대사 출력.
- **힌트 수집 인벤토리 (비밀 수첩)**: 화면 하단/측면에 위치. 맵에서 발견한 힌트(총 3개)가 아이콘 형태로 인벤토리에 수집됨.
- **제한 시간 타이머 화면 표시**: 5:00부터 시작하는 카운트다운 타이머를 화면 상단에 배치하여 긴장감 부여.
- **마우스 커서 커스텀**: 화면 탐색 시 마우스 커서를 '돋보기' 아이콘으로 변경하여 탐색 게임의 느낌 강조.

## 4. 게임플레이 메커니즘 (Gameplay Mechanics)
- **필수 힌트 수집 조건**: 각 스테이지마다 숨겨진 3개의 힌트를 **반드시 인벤토리에 모두 모아야만** 다음 스테이지로 넘어가는 자물쇠/정답 입력창이 활성화됨. (그 전에는 대화창을 통해 "아직 단서가 부족하다"는 메시지 출력)
- **안정성 위주 상호작용 (간소화)**: '명함 드래그' 등의 물리적 제어 대신, 명함을 클릭하면 부드럽게 180도 뒤집히는 CSS 3D 카드 뒤집기 애니메이션을 적용하여 안정성을 확보.
- **페널티 없는 재시도**: 정답 입력창에서 오답을 입력해도 추가 시간 차감 페널티 없이 무제한 재도전 가능.

## 5. 스테이지 구성 (Stage Details)
### STAGE 1: 텅 빈 탐정 사무소
- **배경**: 책상, 노트북, 달력, 마시다 만 커피가 있는 사무실 전경
- **기믹**: 
  1. 달력 클릭 ➔ 특정 날짜(조수의 생일 등) 강조 표시 (힌트 1 획득)
  2. 책상 위 명함 클릭 ➔ 3D 효과로 명함이 뒤집히며 숨겨진 숫자 노출 (힌트 2 획득)
  3. 주변 사물(커피잔 등) 조사 ➔ 수첩 일부 단서 (힌트 3 획득)
- **해결 패턴**: 3개 힌트를 모두 수집 후 노트북(비밀번호 폼) 클릭 시 입력창 활성화. 달력 날짜와 명함 숫자를 조합하여 6자리 번호 입력. 패스워드 일치 시 STAGE 2로 이동.

### STAGE 2: 범인의 아지트
- **배경**: 어두운 지하 창고, CCTV 모니터 4개, 잠긴 철문, 음향 조작기
- **기믹**:
  1. CCTV 모니터 순차 클릭 ➔ 모니터 속 숫자/이동경로 확인 (힌트 1 획득)
  2. 음향 조작기(녹음기 버튼) 클릭 ➔ 범인의 특징/목소리/멜로디 힌트 텍스트 혹은 오디오 출력 (힌트 2 획득)
  3. 수첩의 마지막 페이지 발견 ➔ 이니셜 K.X 단서 (힌트 3 획득)
- **해결 패턴**: 3개 힌트 수집 후 철문 암호 패드(검색창) 활성화. 용의자 4명 중 조건에 맞는 이름 입력. 
- **엔딩**: 정답 시 수갑 아이콘 애니메이션 효과와 함께 "사건 해결!" 엔딩 컷씬 페이지로 전환.
